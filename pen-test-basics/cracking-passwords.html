Password Cracking 101

John the Ripper is known as the go to password cracking program. It's free and comes standard with Kali Linux. However it is not a point and click use application. All commands must be run through the command prompt. As an example, we will crack the default password that comes with kali, which is "toor," or root backwards.

cp /etc/passwd/ passwd
cp /etc/shadow /shadow
