Wireshark and TcpDump 101

WireShark and tcpdump are programs used to sniff(copy) and anaylze network traffic. They are both open source (free) and comes standard with Kali Linux. However, neither is a point and click application. All tcpdump commands must be run through the command prompt.

By using the tcpdump command "tcpdump -v -w /root/Documents" tcpdump begins to capture and write a tcp packet capture scan to the file, "/root/Documents," with the name of the file tcp0. With the packet capture now active, I opened Firefox Mozilla, then opened the homepage to https://google.com and then searched for something simple, the term 'deer.'
